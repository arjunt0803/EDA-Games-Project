# Python Games Project

## Description

This project is developed using Python and Jupyter Notebook. 
It contains different games and demonstrates Python programming concepts such as loops, functions, conditions, and game logic.

The project is beginner-friendly and designed for learning and practicing Python programming.

## Features
- Interactive games
- Clean Python code
- Beginner friendly
- Easy to understand

## Technologies Used
- Python
- Jupyter Notebook

## How to Run
1. Install Python
2. Install Jupyter Notebook
3. Open the notebook
4. Run all cells

## Learning Outcomes
- Python basics
- Functions
- Loops
- Conditional statements
- Game development basics
